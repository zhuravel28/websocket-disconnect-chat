#  WebSocket Chat — Практична робота  


###  Опис проекту  
Проєкт реалізує веб-чат у режимі реального часу на основі **WebSocket**.  
Додано можливість відключення користувача від чату через звичайний HTTP-запит.

##  Структура проекту  
- `index.html` — клієнтська частина чату  
- `server.js` — WebSocket-сервер + HTTP маршрут `/disconnect`  
- `package.json` — залежності Node.js  
- `node_modules` — бібліотеки  
- `README.md` — опис проекту
